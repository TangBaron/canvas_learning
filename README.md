# Canvas_learning

该系列是canvas的练手项目，利用canvas完成简单的图形绘制(矩形等基本图形)->简单的实例(时钟等图形)->小游戏(五子棋，flappy bird)/海报绘制等基本的场景。

该项目的背景是帮助canvas的初学者在初学canvas过程中可以通过实际的案例快速上手并进一步开发出更有趣的项目！

## Canvas基础知识

关于canvas的基础知识大家应该都有一定的了解，这里不加赘述，想学习的同学可以参考
